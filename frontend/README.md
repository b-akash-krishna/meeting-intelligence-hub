# Frontend

Next.js dashboard for the Meeting Intelligence Hub MVP.

## Features
- Transcript upload UI
- Action item and decision dashboard
- Sentiment timeline and speaker tone cards
- CSV export from the dashboard
- Chat drawer for cited meeting questions

## Run
```bash
npm install
npm run dev
```

Open:
- `http://localhost:3000`

The frontend expects the FastAPI backend at:
- `http://localhost:8000`

## Quality Check
```bash
npm run lint
```
